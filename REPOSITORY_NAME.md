# 推奨リポジトリ名

GitHubリポジトリとして公開する際の推奨名称：

## 最推奨

### `jaxa-earth-vrchat-terrain`

**理由**:
- ✅ 検索されやすい（jaxa, earth, vrchat, terrain）
- ✅ 機能が明確（VRChat向け地形生成）
- ✅ 短くて覚えやすい
- ✅ ハイフン区切り（GitHubベストプラクティス）

## その他の候補

### `satellite-terrain-vrchat`
- より一般的な名前
- JAXAに限定されない
- 検索されやすい

### `earth-terrain-vrchat`
- シンプルで覚えやすい
- 検索されやすい

### `jaxa-satellite-vrchat`
- JAXAを強調
- シンプル

## リポジトリ説明（GitHub用）

推奨する説明文：

```
Transform JAXA satellite data into VRChat worlds! Generate 3D terrain from Earth observation data for Blender/Unity/VRChat. Works with Cursor/Codex IDE.
```

または日本語版：

```
JAXAの衛星データからVRChatワールドの地形を自動生成！Blender/Unity/VRChatで使える3D地形を生成するMCPサーバー
```

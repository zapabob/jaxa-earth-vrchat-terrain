# サンプルと例

このディレクトリには、JAXA Earth API MCP Serverの使用例が含まれています。

## ディレクトリ構成

```
examples/
├── README.md (このファイル)
├── basic_usage.py          # 基本的な使用例
├── vrchat_workflow.py      # VRChat向けワークフローの例
└── output/                 # 生成されたサンプルデータ（.gitignoreで除外）
    ├── heightmaps/         # 高度マップのサンプル
    ├── textures/           # テクスチャのサンプル
    └── terrain_data/       # 地形データのサンプル
```
